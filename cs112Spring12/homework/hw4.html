CS112: Homework 4

 For this assignment I want you to create a program that prompts the user to enter a number the number 99 and keeps badgering the user until they do enter 99 (this will require the use of a while loop). Please have the badgering (insulting) vary depending on whether the user enters a number that is larger or smaller then 99 (this will require an if statement).

Once the user enters 99 have your program print all the lyrics for the some 99 bottles of beer on the wall (again, this will use a while loop).

To make your lives easier I would recommend writing a function to do each of the above. Good luck.

Turning it in:

  1. Create a zipped archive of all the files associated with this assignment and name it with your name and the homework assignment (for example paulDicksonHW4.zip).
  2. Place a copy of this zipped archive in the handin folder for the class on urza.
  3. Send an e-mail saying you've turned it in with the following format:
    To: pedcs-at-hampshire.edu
    Subject: 112-HW4: Your name

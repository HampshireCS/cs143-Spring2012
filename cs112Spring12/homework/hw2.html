CS112: Homework 2

 This assignment is another for getting your feet wet. Create a program that sets a number of variables, does math on them, prints them, etc. Basically, use everything we went over in class in your program. As an example of what you could do, you could create a variable and then add to it, multiply it, etc and print it's value at each step. Please try to be creative though and to create an interesting program.

The purpose of this assignment is to make sure that you are comfortable with programs, variables, and basic math.

Turning it in:

  1. Create a zipped archive of all the files associated with this assignment and name it with your name and the homework assignment (for example paulDicksonHW2.zip).
  2. Place a copy of this zipped archive in the handin folder for the class on urza.
  3. Send an e-mail saying you've turned it in with the following format:
    To: pedcs-at-hampshire.edu
    Subject: 112-HW2: Your name

CS112: Homework 1

 There are two purposes for this assignment. First, to make sure that each of you figures out how you are going to work on assignments for this class be it either working on lab computers or getting all of the appropriate software installed on your own computer. If you decide to use your own computer please visit the software link to find out what the software is. This installation may take a while. Second, this assignment is designed to give you a first task of programming on your own so that you can start to get comfortable with it.

Create a program that when executed writes multiple lines of text that say anything you want them to.

Turning it in:

  1. Create a zipped archive of all the files associated with this assignment and name it with your name and the homework assignment (for example paulDicksonHW1.zip).
  2. Place a copy of this zipped archive in the handin folder for the class on urza.
  3. Send an e-mail saying you've turned it in with the following format:
    To: pedcs-at-hampshire.edu
    Subject: 112-HW1: Your name
